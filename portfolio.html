Portfolio

Hikeable

A simple app where you can search by location for hiking/mountain biking trails. This was the first collaborative project I worked on. It was the first time working with ajax for API calls and Zurb Foundation (a flexbox framework) for styling.

LanceUp

A place for stakeholders and web devs to post projects. The idea is stakeholders can build there own team of devs from a pool of candidates based on matching criteria. This was my second group project in which we created a small database to hold our pool of candidates.

Traveld

A travel itinerary builder. I'm continuing to work on this to add social capabilities so people can view others' trips to get ideas and leave comments on trips. We built it using a Mongo database which I am revising to a relational database.
